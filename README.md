## Overview

A simple and efficient Python tool for real-time visualization of Prague integrated transport departure boards. Designed to run on **Raspberry Pi** with output to a **mobile phone/tablet**, serving as a dedicated information display panel.

## Features

- Real-time departure data via Golemio API
- Responsive mobile interface
- Lightweight Raspberry Pi deployment
- Clean, user-friendly dashboard

## Installation

```bash
git clone git@github.com:lordmalina/mhd_praha.git
pip install -r requirements.txt
```

## Usage

```bash
python app.py
```

Access the dashboard on your device at `http://<raspberry-pi-ip>:8000`
